
COMPILAZIONE/PUBLISH DEL SERVIZIO

	Preparazione dei files per l'installazione 
	(da eseguire nell'ambiente di sviluppo)

	1. Pubblicare in una cartella 
	   (ad esempio ..\bin\Release\net6.0\publish)

	2. Copiare/spostare i file della cartella sopra
	   nella cartella di installazione (questa cartella
	   verrà specificata nel comando di creazione del
	   servizio e da questa verrà caricato il servizio
	   ogni volta che viene avviato).
	   Tralasciare nella copia i file pdb e i file localizzati (it, ru, en, ecc)
	   Importante: La cartella runtimes va inclusa 

INSTALLAZIONE ED UTILIZZO (WINDOWS)

	Il servizio è stato chiamato CtlServices

	1. Per installare aprire PowerShell come AMMINISTRATORE
	   ed eseguire i seguenti comandi:

	   # crea il servizio ( se necessario cambiare il path del file eProcurementNext.Services.exe )

	   sc.exe create "CTLServices" displayname="CTL Service" start=delayed-auto binpath="E:\PortaleGareTelematiche\Server\EprocNext.Services\eProcurementNext.Services.exe" 

	   # aggiunge descrizione servizio

	   sc.exe description "CTLServices" "servizio per eseguire in background i processi dell'applicazione web eProcurementNext"

	2. Per utilizzare il servizio, aprire il programma di utilità "Servizi" del
	   sistema operativo e cercare il servizio con nome CtlServices.
	   Il servizio potrà essere avviato o arrestato come qualsiasi altro servizio
	   di Windows.

	3. Per disinstallare aprire PowerShell come al punto 3 ed eseguire il comando:

	   sc.exe delete "CtlServices"

	4. Per aggiornare ripetere le operazioni descritte sopra.
